== Description ==
 
This plugin helps administrators of the site see their environment on which the site is currently running that includes WordPress environment, tha database it requires, Server Environment and the plugins installed and activated on the site.
If you need additional help you can contact us for [Custom Development](https://wbcomdesigns.com/contact/).
 
== Installation ==
 
1. Upload the entire "wp-system-log" folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
 
== Frequently Asked Questions ==
 
= How can we see the details this plugin provides? =
 
You'll get a menu page namely, "WP System Log" mostly at the bottom of the admin menus which will fetch the complete details.
